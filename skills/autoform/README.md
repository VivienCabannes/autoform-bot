# autoform

Mathlib & Lean 4 conventions for writing formalization-quality code. Distilled from 94k PR review comments and 165k Zulip messages from the Mathlib community.

Covers proof style, naming, types, tactics, simp conventions, API design, code style, and common pitfalls.

## Usage

```
/autoform
```

Activates automatically when you're writing Lean 4 code or working with Mathlib.
